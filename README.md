# Trylma Game (not completed)

## Overview
Trylma, also known as Chinese Checkers, is a strategy board game. This project aims to implement the game in Java.

## Project Status
This project is currently a work-in-progress and not yet completed.

## Requirements
- Java 8 or higher
- Maven

## Instructions

### Clone the Repository
```sh
git clone https://github.com/unseen2004/Trylma.git
cd Trylma
```

## Compile the Project
To compile the project using Maven, run the following command:
```sh
mvn clean install
```
## Run
To run the server, use the following command:
```sh
mvn exec:java -Dexec.mainClass="com.example.trylma.Server"
```
To run the client, use the following command:
```sh
mvn exec:java -Dexec.mainClass="com.example.trylma.Client"
```
